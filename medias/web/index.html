About The Project

Fandem allows you to transfer files transfer in Peer to Peer (P2P), meaning that the data sent goes directly to the other device, without any intermediate server in betweenThis kind of transfer allows you to keep your data private

The transfers are performed using your Wi-Fi (or smartphone's data), so they are a lot more faster than with bluetooth!

This application works with both Android and Desktop, meaning that ou can send/receive a file from a smartphone to a computer, and vice-versa

How to use

1. Make sure both devices are on the same network

In order for the P2P transfer to work, both devices need to be under the same Wi-Fi, Or if one of the devices is a smartphone, you can share data (mobile hotspot) and have the other device connect to it

2. (Sender) Select a file to share

The sender should select a file. You can do it from the app, or by sharing a file on Android, and then select Fandem. Once it is done, the sender peer will wait for a connection. If one of the devices is a smartphone, you can share data (mobile hotspot) and have the other device connect to it

3. (Receiver) Find the sender peer

The app can automatically detect the sender peer, it will be proposed to you once the sender selected a file

4. Let the transfer begin

The transfer can now begin. Enjoy sharing files!
